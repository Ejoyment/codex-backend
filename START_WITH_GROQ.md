# 🚀 Start AI Pair Programming with Groq

## Quick Start (30 seconds)

### 1. Start Backend
```bash
START_AI_PAIR.bat
```

### 2. Open Dashboard
http://localhost:5500/dashboard.html

### 3. Click "AI Pair"
Look for the chat bubble icon in the sidebar

### 4. Select Repository
Choose a GitHub repo from the dropdown

### 5. Start Chatting!
Type your question and press Enter
